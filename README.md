Linguagens:

Requisitos:
- XAMPP / WAMP / Laragon
- PHP 7+

Como rodar:

1. Coloque a pasta no htdocs (XAMPP)
2. Inicie Apache e MySQL
3. Crie o banco usando o SQL fornecido
4. Acesse:
http://localhost/formulario

Segurança:
- password_hash() (criptografia)
- Prepared Statements (anti SQL Injection)
- Validação de dados (frontend + backend)

obs:
As senhas exibidas estão criptografadas (hash), não são as reais.
